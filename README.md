# 🐾 Pet Adoption UI Frontend

A responsive and modern pet adoption user interface built using **HTML**, **Tailwind CSS**, and **JavaScript**. The application allows users to browse pets, search by name, and filter pets based on categories for a smooth adoption experience.

## ✨ Features

* Responsive navigation bar
* Attractive hero section
* Pet listing cards
* Search functionality
* Category-based filter
* Responsive design for different screen sizes
* Clean and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS
* JavaScript (ES6)

## 📂 Project Structure

```text
├── assets/
├── index.html
├── styles.css
├── script.js
└── README.md
```

## 🚀 Getting Started

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
